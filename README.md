# Discord ぼっち用bot

Discordのbotを操作することができるpythonのAPI「discord.py」でぼっち用に便利なぼっとを作成する。

## DEMO

![botti_demo](./botti_demo.png)

## Requirement

* poetry
* Python 3.8.10

## Installation

```bash
$ poetry install
```

## Note

* python,gitのお勉強用のプロジェクト。
* できるだけ便利なぼっとを作りたい。
* 最終的にはPaasの無料プランで24時間365日稼働させるのが目標。
